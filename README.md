# BlogApp
 A Mern Stack BlogApp where you can create your own blog, update it and delete it and other features like comment on the blogpost and like and dislike as well . 
 
 Technology Used : 
 Frontened : ReactJS  
 Backened : NodeJs/ExpressJS 
 DataBase : Mongodb
